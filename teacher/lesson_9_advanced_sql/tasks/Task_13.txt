После того как вы заполнили базу данных тестовыми данными
при помощи SQL команды insert пришло время поэксперементировать
с SQL командой update.

Создайте для таблиц books/readers/reader_books различные
варианты команды update:
  - изменение всех данных в таблице;
  - изменение данных по id;
  - изменение одного свойства (не id);
  - изменение всех свойств;
  - ваши варианты SQL команды update.

Составленные вами SQL команды проверьте на реальной базе данных
выполнив их.

Написанные команды сохраните в файле dbDataUpdateScript.sql.
